# 时间管理故事

## Concept 
擅长滚雪球的从比较容易的成功，而成功的关键在于构建自己的核心系统 。 

## 时间管理成功步骤
1、 种子 -- 寻找时间黑洞  
2、 树苗 -- 四象限规划  
3、 成长 -- 要事优先，应对突发，（简单明确，即时满足）  
4、 开花 -- 养成习惯  
5、 结果 -- 实践，落地  
6、 收获 -- 高效率，漫生活  

## 方法工具
1、 思维导图梳理框架
2、 甘特图控制进度
3、 九宫格平衡人生

## 杂项
收集的关键是清空大脑。
